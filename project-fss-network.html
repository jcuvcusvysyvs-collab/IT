Фонд социального страхования РФ

Создание универсальной высокопроизводительной отказоустойчивой корпоративной сети

Москва, 2019

01

Задача

В рамках концепции внедрения новых программных сервисов у заказчика возникла необходимость в модернизации локальной вычислительной сети здания. Целями модернизации являлись:

  • увеличение общей производительности сетевой инфраструктуры за счёт перехода на магистральные соединения 10 Гбит/с;
  • повышение уровня отказоустойчивости сетевой инфраструктуры;
  • подготовка инфраструктуры к внедрению сервисов, использующих технологию PoE;
  • внедрение решений по обеспечению безопасного доступа пользователей в сеть Интернет;
  • замена устаревшего, снятого с производства оборудования.
02

Решение

Была произведена поставка оборудования, подготовлена необходимая документация, выполнено внедрение решения.

03

Результат

В результате работ по модернизации создана универсальная, производительная и отказоустойчивая сетевая инфраструктура, соответствующая требованиям для дальнейшего развития информатизации производственного процесса заказчика.

Оборудование:

  • Уровень доступа: стеки коммутаторов Cisco Catalyst 9300;
  • Защита и контроль WEB-трафика: Cisco Web Security Appliance S395;
  • Управление защитой контента: Cisco Security Management Appliance M195.
