ФКУ «Росдоринформсвязь»

Замена системы резервного копирования для 21 ИС без остановки работы.

Москва, 2023

01

Задача

Модернизация системы резервного копирования с целью обеспечения необходимой ёмкости хранения и импортозамещения для устранения рисков, связанных с использованием оборудования и ПО иностранных производителей.

02

Решение

В 2023–2024 гг. выполнены работы по успешному построению модернизированной системы резервного копирования на основе следующих отечественных решений:

  • Кибер Бэкап: ПО версии 16.5 (редакция Расширенная);
  • Astra Linux: ОС серверов резервного копирования 1.7 Воронеж;
  • YADRO Tatlin.Unified.SE: СХД для размещения резервных копий.
03

Результат

Выполнена интеграция для обеспечения защиты и настроены политики резервного копирования, которые обеспечили защиту более 130 ТБ боевых данных, включая следующие системы:

  • 32 хоста виртуализации VMware 6.5 и 6.7 с более 250 ВМ;
  • 30 агентов для защиты объектов внутри ОС и приложений (включая СУБД PostgreSQL), почтовая система предприятия на базе MS Exchange Server в конфигурации DAG с возможностью резервирования отдельных почтовых ящиков.

Проводится ежеквартальное тестирование восстановления из резервных копий полного набора данных каждой из продуктивных информационных систем для обеспечения гарантированного возобновления работоспособности.
